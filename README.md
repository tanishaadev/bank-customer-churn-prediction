# Bank Customer Churn Prediction
Predicting customer churn for a European bank using machine learning on 10,000 customer records.

## Key Results
- Random Forest: F1-Score = 0.89, AUC = 0.95
- Gradient Boosting: F1-Score = 0.88, AUC = 0.94

## Techniques Used
- Exploratory Data Analysis & Visualization
- SMOTEENN for class imbalance handling
- Random Forest & Gradient Boosting classifiers
- Hyperparameter tuning
- ROC-AUC Curve & Confusion Matrix evaluation
- Feature Importance Analysis

## Dataset
Bank Customer Churn Dataset - Kaggle (10,000 records, 10 features)
